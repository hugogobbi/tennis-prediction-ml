# tennis-prediction-ml
Machine learning models to predict tennis results

Objective
The objective is to built machine learning models that predicts the outcome of tennis matches based on past results and players' characteristics.

Data
Past match result data is found in the website http://tennis-data.co.uk/alldata.php with outcomes covering several years. A short description of each variable can be found here : http://www.tennis-data.co.uk/notes.txt
The match outcomes data is combined with player specific characteristics web scrapped from the players profile pages in the ATP website. The main characteristics are birthday (age), hand, backhand and height.
